Glosario Técnico del Programador

API
Interfaz de programación que permite la comunicación entre aplicaciones.
Framework
Estructura de soporte que facilita el desarrollo de aplicaciones con una base reutilizable de código.
Debugging
Proceso de identificar y corregir errores en el código fuente.
Repositorio
Ubicación central donde se almacena y gestiona el código fuente (por ejemplo, en GitHub).
Frontend
Parte visual de una aplicación web con la que interactúa el usuario (HTML, CSS, JavaScript).
Backend
Lógica del lado del servidor que gestiona la base de datos, autenticación, y reglas de negocio.
Compilador
Programa que traduce el código fuente a un lenguaje de máquina ejecutable.
